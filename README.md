# Bank-Telemarketing-Success-Prediction-using-XGBoost-Model
Predicting Term Deposit Subscriptions Using Machine Learning
This project explores the Bank Marketing Dataset to classify customer responses (“yes”/“no”) to telemarketing campaigns. The workflow includes preprocessing, visual EDA, handling class imbalance, training multiple ML models (Logistic Regression, Random Forest, XGBoost, etc.), and comparing performance metrics. The full analysis and code are available in the accompanying Kaggle notebook.
